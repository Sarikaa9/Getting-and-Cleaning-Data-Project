# Getting and Cleaning Data: Course Project

## Overview
This project is part of the "Getting and Cleaning Data" course on Coursera. It involves processing and cleaning the Human Activity Recognition dataset to create a tidy dataset for analysis.

## Instructions
1. Clone or download this repository.
2. Place the "UCI HAR Dataset" folder in your working directory.
3. Open `run_analysis.R` in R or RStudio and run the script.
4. The output file `tidy_dataset.txt` will be created in the same directory as the dataset.

## Files
- `run_analysis.R`: R script for processing and cleaning the data.
- `tidy_dataset.txt`: The final tidy dataset.
- `CodeBook.md`: Explains the variables, data, and transformations performed.
